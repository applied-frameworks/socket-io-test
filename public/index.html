Realtime Canvas

Collaborative Drawing Made Simple

Welcome to Realtime Canvas - a powerful collaborative drawing platform where multiple users can create, edit, and share visual content in real-time.

Features

  • Real-time collaborative drawing
  • Multiple drawing tools (rectangle, ellipse, triangle, brush)
  • Color and opacity controls
  • Shape selection and manipulation
  • Live cursor tracking
  • Integrated chat system
  • Secure JWT authentication

This is a public demo page served from the /public directory. No authentication required to view this content.

Powered by Socket.IO, Express, and React
